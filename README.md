# 📝 Task Tracker (MERN Stack)

A full-stack Task Management application built as a technical assignment for the **Full Stack Developer Intern** position at **COLL-EDGE CONNECT**. 

This application provides a seamless, dynamic user experience with real-time DOM updates, responsive design, and robust REST APIs.

## 🚀 Live Links
- **Frontend (Vercel):** task-tracker-mern-m5yffmzh2-vishal-mauryas-projects-dc22849f.vercel.app
- **Backend API (Render):** https://task-tracker-mern-dbkp.onrender.com

##  Features Implemented

###  Mandatory Features (100% Completed)
- **Full CRUD Operations:** Create, Read, Update, and Delete tasks effortlessly.
- **Dynamic UI Updates:** React state management ensures the UI updates instantly without page refreshes.
- **Form Validation:** Prevents empty or invalid task submissions.
- **RESTful APIs:** Clean and modular backend routes connected to MongoDB.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.

###  Bonus Features Added
- **Advanced Filtering:** Filter tasks instantly by their status (All, Pending, In Progress, Completed).
- **Smart Sorting:** Sort tasks dynamically by date (Newest First / Oldest First).
- **Color-Coded Status Badges:** Visual indicators for task progress (Yellow for Pending, Blue for In Progress, Green for Completed).
- **Modern UI/UX:** Built using the latest **Tailwind CSS v4** with smooth scrolling and interactive hover states.
- **Toast Notifications:** Real-time success and error popups for better user feedback.

##  Tech Stack
- **Frontend:** React.js (Vite), Tailwind CSS v4, Axios, React-Hot-Toast
- **Backend:** Node.js, Express.js, CORS, dotenv
- **Database:** MongoDB (Mongoose)
- **Deployment:** Vercel (Frontend) & Render (Backend)
